# Quick Start Guide

## Installation (5 minutes)

### Step 1: Prepare Icons (Optional but Recommended)
1. See `ICONS_INSTRUCTIONS.md` for creating icons
2. Or skip this step for now - extension works without custom icons

### Step 2: Load Extension in Chrome/Edge

1. Open Chrome or Edge browser
2. Navigate to:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
3. Enable **Developer mode** (toggle switch in top-right)
4. Click **Load unpacked**
5. Select the `whatsapp-spam-detector` folder
6. ✅ Extension is now installed!

### Step 3: Test It

1. Open [WhatsApp Web](https://web.whatsapp.com/)
2. Log in to your WhatsApp account
3. Send yourself a test message with spam indicators:
   - Multiple URLs: "Check this: https://example.com and https://test.com"
   - Urgency words: "URGENT! Limited time offer! Click now!"
   - Financial terms: "Verify your bank account immediately"
4. You should see color-coded warnings appear!

## Customization

Click the extension icon in your toolbar to:
- Adjust spam detection threshold (10-70%)
- Enable/disable detection
- Toggle warning visibility
- Manually rescan messages

## Troubleshooting

**Extension not working?**
- Make sure you're on https://web.whatsapp.com/
- Refresh the WhatsApp Web page
- Check that the extension is enabled in `chrome://extensions/`

**No warnings appearing?**
- Try lowering the threshold in extension settings
- Send a test message with obvious spam indicators
- Click "Rescan Messages" in the popup

**Want to disable?**
- Click extension icon → Toggle off "Enable Detection"
- Or disable the extension in `chrome://extensions/`
