# Creating Extension Icons

You need to create three icon files for the extension:

1. `icons/icon16.png` (16x16 pixels)
2. `icons/icon48.png` (48x48 pixels)
3. `icons/icon128.png` (128x128 pixels)

## Quick Options:

### Option 1: Use an Online Tool
- Visit https://www.favicon-generator.org/ or https://favicon.io/

For now, you can use any PNG file renamed to the required names - the extension will still work, just won't show custom icons in the browser toolbar.
